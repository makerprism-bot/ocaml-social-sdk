# Reference Implementations

These are the GitHub repositories we consider strong reference implementations for Bluesky/AT Protocol behavior.

Selection criteria:
- Maintained by the protocol owner or a widely trusted community maintainer
- Meaningful ecosystem adoption (stars/usage)
- Recent repository activity

Research snapshot date: 2026-02-13

## Repositories

- `bluesky-social/atproto` (TypeScript, official)  
  https://github.com/bluesky-social/atproto  
  Canonical implementation from Bluesky; best reference for protocol semantics and endpoint behavior.
  Pinned main SHA: `6ae3cf6131964a7345d5a532dad645b479766ff9`

- `bluesky-social/indigo` (Go, official)  
  https://github.com/bluesky-social/indigo  
  Official Go implementation used for production services; good for server/client protocol edge cases.
  Pinned main SHA: `85cdd0d6871cf13cd0fb15cfa3178eddb49e11aa`

- `MarshalX/atproto` (Python, community)  
  https://github.com/MarshalX/atproto  
  Popular Python SDK with broad ATProto coverage; useful cross-language API design reference.
  Pinned main SHA: `62670ecd00fa082bf9016cd90400b851b1cb3eba`
