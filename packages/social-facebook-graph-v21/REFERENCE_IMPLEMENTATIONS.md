# Reference Implementations

These are the GitHub repositories we consider strong reference implementations for Meta Graph API patterns relevant to this client.

Selection criteria:
- Official maintainer ownership where possible
- Strong adoption and practical usage
- Recent repository activity

Research snapshot date: 2026-02-12

## Repositories

- `facebook/facebook-python-business-sdk` (Python, official)  
  https://github.com/facebook/facebook-python-business-sdk  
  Official Meta SDK with mature auth/request/error handling and broad Graph API usage patterns.

- `facebook/facebook-nodejs-business-sdk` (Node.js, official)  
  https://github.com/facebook/facebook-nodejs-business-sdk  
  Official JavaScript reference for request models, pagination, and token-based flows.

- `facebook/facebook-java-business-sdk` (Java, official)  
  https://github.com/facebook/facebook-java-business-sdk  
  Official Java implementation, helpful for typed object modeling and error-class structure.

- `restfb/restfb` (Java, community)  
  https://github.com/restfb/restfb  
  Long-standing Graph API client with good coverage and practical examples beyond ad-focused flows.
