# Reference Implementations

These are the GitHub repositories we consider strong reference implementations for Instagram Graph API behavior.

Note: Instagram Graph is part of Meta Graph APIs, so trusted references are mostly Graph SDKs rather than Instagram-only SDKs.

Selection criteria:
- Official maintainer ownership where possible
- Strong adoption and practical usage
- Recent repository activity

Research snapshot date: 2026-02-12

## Repositories

- `facebook/facebook-python-business-sdk` (Python, official)  
  https://github.com/facebook/facebook-python-business-sdk  
  Official Meta SDK and a solid reference for Graph auth, paging, and error behavior used by Instagram Graph endpoints.

- `facebook/facebook-nodejs-business-sdk` (Node.js, official)  
  https://github.com/facebook/facebook-nodejs-business-sdk  
  Official JavaScript SDK with practical Graph request/response handling patterns.

- `facebook/facebook-java-business-sdk` (Java, official)  
  https://github.com/facebook/facebook-java-business-sdk  
  Official Java SDK useful for typed request construction and error taxonomy.

- `restfb/restfb` (Java, community)  
  https://github.com/restfb/restfb  
  Widely used Graph API client with reusable implementation ideas for non-ads Graph access.
