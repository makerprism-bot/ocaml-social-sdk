# Reference Implementations

These are the GitHub repositories we consider strong reference implementations for LinkedIn API behavior.

Selection criteria:
- Official LinkedIn developer ownership preferred
- Meaningful adoption and active maintenance
- Useful API modeling and auth examples

Research snapshot date: 2026-02-12

## Repositories

- `linkedin-developers/linkedin-api-python-client` (Python, official)  
  https://github.com/linkedin-developers/linkedin-api-python-client  
  Official LinkedIn Python client; primary reference for current OAuth/API interaction patterns.

- `linkedin-developers/linkedin-api-js-client` (JavaScript, official)  
  https://github.com/linkedin-developers/linkedin-api-js-client  
  Official JavaScript client with clear request construction and endpoint usage conventions.

- `linkedin-developers/java-sample-application` (Java, official sample)  
  https://github.com/linkedin-developers/java-sample-application  
  Not a full SDK, but a trusted reference for end-to-end LinkedIn API flows in Java.
