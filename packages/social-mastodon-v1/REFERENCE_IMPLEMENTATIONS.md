# Reference Implementations

These are the GitHub repositories we consider strong reference implementations for Mastodon API behavior.

Selection criteria:
- Strong ecosystem trust and adoption
- Active maintenance
- Clear API coverage and practical examples

Research snapshot date: 2026-02-12

## Repositories

- `halcy/Mastodon.py` (Python, community standard)  
  https://github.com/halcy/Mastodon.py  
  Most established Python Mastodon API wrapper; good reference for endpoint coverage and pagination.

- `mastodon/mastodon-api` (Ruby, project-owned)  
  https://github.com/mastodon/mastodon-api  
  Maintained in the Mastodon org; useful for API conventions close to the upstream project ecosystem.

- `mattn/go-mastodon` (Go, community)  
  https://github.com/mattn/go-mastodon  
  Widely used Go client with practical patterns for auth, timelines, and streaming-adjacent APIs.
