# Reference Implementations

These are the GitHub repositories we consider strong reference implementations for Pinterest API v5 behavior.

Selection criteria:
- Official Pinterest ownership preferred
- Reasonable ecosystem trust
- Useful endpoint/auth/request modeling

Research snapshot date: 2026-02-12

## Repositories

- `pinterest/pinterest-python-sdk` (Python, official)  
  https://github.com/pinterest/pinterest-python-sdk  
  Official Pinterest SDK with hand-authored usage patterns and examples.

- `pinterest/pinterest-python-generated-api-client` (Python, official generated)  
  https://github.com/pinterest/pinterest-python-generated-api-client  
  Official OpenAPI-generated client; useful reference for endpoint surface and schema alignment.

## Notes

- The public Pinterest ecosystem has fewer trusted, actively maintained SDKs in non-Python languages.
- For additional language references, generated clients from Pinterest OpenAPI definitions are a practical fallback.
