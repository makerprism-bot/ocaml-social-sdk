# Reference Implementations

These are the GitHub repositories we consider strong reference implementations for TikTok API behavior relevant to this client.

Selection criteria:
- Official TikTok ownership preferred
- Practical API coverage and examples
- Recent activity

Research snapshot date: 2026-02-12

## Repositories

- `tiktok/tiktok-business-api-sdk` (Python, official)  
  https://github.com/tiktok/tiktok-business-api-sdk  
  Primary official SDK reference for TikTok Business API auth and request/response patterns.

- `tiktok/tiktok-research-api-wrapper` (Python/R, official)  
  https://github.com/tiktok/tiktok-research-api-wrapper  
  Official wrapper for TikTok Research API flows; useful for request structure and pagination ideas.

## Notes

- Public, high-trust TikTok SDK options are relatively limited and often API-scope specific.
- We prioritize official repos even when language coverage is narrower.
