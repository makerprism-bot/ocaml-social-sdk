# Reference Implementations

These are the GitHub repositories we consider strong reference implementations for Twitter/X API v2 behavior.

Selection criteria:
- Official maintainer ownership where available
- Strong adoption and practical usage
- Recent repository activity

Research snapshot date: 2026-02-12

## Repositories

- `xdevplatform/twitter-api-typescript-sdk` (TypeScript, official)  
  https://github.com/xdevplatform/twitter-api-typescript-sdk  
  Official TypeScript SDK; key reference for current endpoint typing and request patterns.

- `xdevplatform/twitter-api-java-sdk` (Java, official)  
  https://github.com/xdevplatform/twitter-api-java-sdk  
  Official Java SDK and a good reference for generated models and error handling behavior.

- `tweepy/tweepy` (Python, community standard)  
  https://github.com/tweepy/tweepy  
  Highly adopted Python client with mature OAuth and API usage patterns across v1.1/v2 surfaces.
