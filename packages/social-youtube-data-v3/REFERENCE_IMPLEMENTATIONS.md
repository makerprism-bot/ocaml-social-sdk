# Reference Implementations

These are the GitHub repositories we consider strong reference implementations for YouTube Data API v3 behavior.

Selection criteria:
- Official Google ownership
- High ecosystem adoption
- Active maintenance

Research snapshot date: 2026-02-12

## Repositories

- `googleapis/google-api-nodejs-client` (Node.js, official)  
  https://github.com/googleapis/google-api-nodejs-client  
  Official Node.js client; strong reference for discovery-based request patterns and auth flows.

- `googleapis/google-api-python-client` (Python, official)  
  https://github.com/googleapis/google-api-python-client  
  Official Python client with extensive YouTube Data API examples and mature OAuth handling.

- `googleapis/google-api-go-client` (Go, official)  
  https://github.com/googleapis/google-api-go-client  
  Official Go discovery-based client; useful for typed service construction and pagination behavior.

- `googleapis/google-api-java-client-services` (Java, official generated)  
  https://github.com/googleapis/google-api-java-client-services  
  Generated Java service bindings for Google APIs, including YouTube Data API v3 models/endpoints.
